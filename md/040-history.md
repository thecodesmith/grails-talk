## History

### Grails 1.0 released February 2008

Main Features:

* GORM: An Object-Relational Mapping DSL for advanced Mappings
* Support for easy to use Filters: 
    `User.findAll { user -> user.name == 'Joe' }`
* REST support with JsonBuilder and XmlBuilder
